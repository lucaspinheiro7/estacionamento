# Sistema para gerenciamento de um estacionamento

## Requisitos Funcionais:

* Registrar entrada de veículos;
* Listar veículos no pátio;
* Calcular total a pagar pelo:
	preço/hora x quantidade de horas no estacionamento;
* Remover veículos do pátio;

## tecnologias utilizadas:

* Spring Boot 
* Spring Boot DevTools
* Spring Tool Suite
* Spring MVC
* Spring Data JPA
* Thymeleaf
* Bootstrap
* Maven
* PostgreSQL
* Tomcat
* JPA e Hibernate
* Bean Validation
* JQuery
* Joda-Time
